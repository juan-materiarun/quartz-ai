## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn
- Google AI API key (free tier available)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd quartz-ai
```

2. Install dependencies:
```bash
npm install
```

3. Set up your Google AI API key:
   - Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
   - Create a new API key (free tier available)
   - Create a `.env.local` file in the root directory:
   ```env
   GOOGLE_GENERATIVE_AI_API_KEY=your_api_key_here
   ```

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

## Features in Detail

### The Audit Input
- Clean, professional URL input field
- Code snippet textarea with syntax-friendly monospace font
- Tab-based interface for switching between Live Site and Code Snippet modes
- Prominent "Start Audit" button with bank-grade styling

### Processing State
- Animated progress bar
- Real-time status logs
- Step-by-step analysis indicators
- Technical scanning animation

### AI Auditor
Powered by Google Gemini 1.5 Flash, analyzes:
- **HTML/UI Structure**: Broken tags, accessibility issues, semantic HTML
- **Input Security**: Form validation, XSS vulnerabilities, CSRF protection
- **Performance**: Heavy assets, slow scripts, optimization opportunities

### Report Generator
- **Passed Tests**: Green cards showing what's working
- **Defects**: Color-coded by priority (Critical/Medium/Low)
- **Test Script**: Ready-to-use Playwright test code generated by AI

## Customization

### Styling
Modify `app/globals.css` and `tailwind.config.ts` to customize the theme.

### Audit Logic
Update `app/api/audit/route.ts` to customize the AI prompt or add additional analysis criteria.
